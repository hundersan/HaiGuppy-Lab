# HaiGuppy Lab
## 艾爾銀短身育種研究計畫

---

# 一、研究宗旨

HaiGuppy Lab 致力於建立孔雀魚育種數據化管理模式。

透過：

> 問題提出 → 計畫建立 → 實驗執行 → 結果驗證

四階段流程，將傳統仰賴經驗與運氣的繁殖模式，逐步轉化為可記錄、可追蹤、可驗證的育種系統。

目前研究主軸以：

**艾爾銀短身雙劍系統（Ail Silver Short Body Sword）**

為核心，並同步進行體色、尾型、白子化與日本藍等相關導入計畫。

# 五、研究計畫格式範本

## ◆實驗計畫 2026-0001

### ◇計畫名稱

短身日本藍改造計畫

### ◇參與品系

JBDS（公） × SB-AIS-R（母）

### ◇實驗覽圖

1. JBDS（公） × SB-AIS-R（母）產出 F1
2. F1（公） × SB-AIS-R（母）產出 F2
3. F2 將出現實驗目的
4. F2 互交留下穩定性數據

### ◇研究問題

日本藍基因是否可穩定導入艾爾銀短身系統？

---

### ◇當前狀態

已結案

### ◇建立日期

2025/XX/XX

### ◇結案日期

2026/05/31

---

### ◇執行結果

1. 成功取得 F1 子代。
2. F1 成魚後出現多種安德拉斯表現型。
3. 品系穩定性不足。
4. 停止後續繁殖。

### ◇保留成果

- 艾爾銀日本藍公魚數尾
- 艾爾銀安德拉斯公魚數尾

### ◇結論

安德拉斯來源基因過於複雜，不適合作為短身日本藍固定化主力血統。
